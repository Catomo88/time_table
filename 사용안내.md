# 두 아이 시간표 관리 — 사용 안내 (2026.03 기준)

## 파일
- **시간표.xlsx** — 인쇄용. 시트 4개:
  - `첫째 주간표` / `둘째 주간표` : 요일×시간 색상 타임테이블 (가로 1페이지 인쇄 설정 완료)
  - `일정(편집용)` : 일정을 고치는 곳
  - (필요 시 점검용 계산도 추가 가능)
- **시간표_대시보드.html** — 모바일 공유용. 요일 탭을 누르면 첫째·둘째를 큰 색상 카드로 표시.
- 원본 사진: 시간표_초안.png

## 바뀔 때
1. `일정(편집용)` 시트에서 줄을 추가·수정·삭제 (구분/요일/분류는 목록 선택).
2. "주간표·대시보드 갱신해줘"라고 말씀하시면 색상 타임테이블과 모바일 화면을 다시 그려 드립니다.

## 색상
정규수업·어린이집(파랑) / 방과후(초록) / 학원(주황) / 이동·돌봄(노랑) / 빈 시간(흰칸).
아이 구분: 첫째(초록 헤더) · 둘째(파랑 헤더).
`*` 표시 = 사진에 끝나는 시간이 없어 약 50분으로 추정 (확인 필요).

## 인쇄 / 공유
- 엑셀: 각 주간표 시트를 가로로 1페이지 인쇄.
- 모바일: 대시보드(html)를 카톡·메일로 전송 → 브라우저로 열람, 또는 "인쇄/PDF 저장".
